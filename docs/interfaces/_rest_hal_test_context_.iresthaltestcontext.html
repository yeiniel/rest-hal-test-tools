Interface IRestHalTestContext

Ava Contextual Test Context specific for the REST HAL tools.

A specialized Contextual Test Context is provided to specify the required elements that are expected to be provided by the Ava test context. This elements are: the URL of the resource under test and the SuperTest agent to use.
